# ⚽ Futsal Timekeeper PRO

Aplikasi Scoreboard dan Timekeeper Futsal berbasis web yang didesain khusus untuk memenuhi standar pertandingan futsal, terinspirasi dari kebutuhan teknis **Wasit Lisensi Level 2 Asprov Banten**.

![Futsal Timekeeper](https://img.shields.io/badge/Status-Development-orange)
![License](https://img.shields.io/badge/Wasit-Level%202-blue)

## 🌟 Fitur Utama

- **Real-time Match Timer**: Pengaturan waktu fleksibel (10, 15, atau 20 menit).
- **Foul Tracking**: Indikator peringatan otomatis ketika tim mencapai 5 kali foul (titik penalti kedua).
- **Timeout Management**: Hitung mundur 60 detik otomatis untuk timeout tim.
- **Keyboard Shortcuts**: Kontrol penuh tanpa mouse untuk respon cepat di meja juri.
- **Fullscreen Mode**: Tampilan bersih untuk output ke proyektor atau monitor besar.

## ⌨️ Shortcut Keyboard

Gunakan tombol berikut untuk kontrol yang lebih cepat dan presisi:

| Tombol | Aksi |
| :--- | :--- |
| `Space` | Start / Pause Timer |
| `H` | Tambah Goal Home |
| `A` | Tambah Goal Away |
| `F` | Tambah Foul Home |
| `J` | Tambah Foul Away |
| `1` | Panggil Timeout Home |
| `2` | Panggil Timeout Away |
| `R` | Reset Pertandingan |
